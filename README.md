# Movie-Rating_Prediction
Machine Learning project that predicts IMDb movie ratings using regression models and feature engineering.
# Movie Rating Prediction with Python

## Project Overview
This project predicts IMDb movie ratings using Machine Learning techniques. The model analyzes movie features such as genre, duration, director popularity, actor frequency, movie age, and votes.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Linear Regression
- Ridge Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Features Engineered
- Genre Encoding
- Movie Age
- Log Votes
- Director Frequency
- Actor Frequency
- Duration

## Exploratory Data Analysis
- Rating Distribution
- Genre-wise Average Ratings
- Ratings by Decade
- Duration vs Rating Analysis
- Director Performance Analysis

## Results
Gradient Boosting and Random Forest models achieved the best prediction performance.
